\doxysubsection{Opis szczegółowy}
Ładuje sceny startowe gry przy uruchomieniu aplikacji. 

Skrypt sprawdza, czy scena inicjalizacyjna ({\ttfamily Init\+Scene}) jest załadowana. Jeśli nie — ładuje ją jako scenę dodatkową (Additive), a następnie ładuje scenę główną ({\ttfamily \doxylink{class_main_menu}{Main\+Menu}}) jako podstawową (Single).

Używany jako punkt wejściowy aplikacji.
